# Changelog

All notable changes to this project are documented here.

## [1.0.0] - 2026-05

### Added
- Multi-provider AI support: Ollama (local), Google Gemini, Groq, OpenRouter
- Automatic PDF table-of-contents detection and chapter navigation
- Dual PDF viewer (extracted text + rendered page images)
- Seven standard teaching modes (Explain, Teach, Summarize, Simplify,
  Code Walkthrough, Quiz, Interview Prep)
- Full VARK learning system:
  - Style-detector quiz
  - Visual, Auditory, Read/Write, Kinesthetic, and Omni modes
- Visual rendering: mermaid diagrams (online + local graphviz fallback),
  matplotlib charts, ASCII diagrams
- Export tools: Markdown / HTML / text notes, Anki flashcards, HTML mindmaps
- Text-to-speech playback for auditory learning
